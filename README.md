# Exploratory Data Analysis (EDA) on Student Mental Health Dataset

## Objective
Analyze student mental health data to discover patterns, trends, and relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Statistical Summary
- Missing Value Analysis
- Gender Distribution
- Depression Analysis
- Anxiety Analysis
- Panic Attack Analysis
- Age Distribution
- Correlation Heatmap

## Key Findings
- Gender distribution was analyzed.
- Mental health factors were explored.
- Correlations between variables were identified.
- Important insights were obtained through visualizations.

## Conclusion
EDA helped uncover patterns and trends in student mental health data.
